Kullanım Talimatları:

Bu sayfayı AppCreator24'te kullanmak için aşağıdaki adımları takip edin:

  1. Bu HTML dosyasını GitHub deposunuzda index.html olarak kaydedin
  2. GitHub Settings > Pages bölümünden GitHub Pages'i etkinleştirin
  3. Oluşan GitHub Pages URL'ini AppCreator24'te iframe olarak kullanın
  4. Örnek: <iframe src="https://kullaniciadi.github.io/english-audio/" style="width:100%;height:100vh;border:none;"></iframe>
